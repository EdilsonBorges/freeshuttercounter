FreeShutterCounter
macOS application that shows Canon EOS DSLR shutter count
Application is absolutely free. No charge or payments required.

Supported cameras

1D C, 1D X, 1D Mark IV, 7D Mark II, 7D, 5D Mark III, 5D Mark II, 6D, 70D, 60D, 50D
700D (Rebel T5i), 650D (Rebel T4i), 600D (Rebel T3i), 550D (Rebel T2i), 500D (Rebel T1i), 100D (Rebel SL1), 1200D (Rebel T5), 1100D (Rebel T3), 1000D (Rebel XS)
Unsupported: 1D X Mark II, 5DS, 5DS R, 5D Mark IV, 5D, 80D, 750D (Rebel T6i), 760D (Rebel T6s), 1300D (Rebel T6)

Download

freeshuttercounter-1.2.2.dmg (1.6 Mb)
Compatible with macOS 10.7-10.12

Usage

  • copy app into the /Applications folder
  • switch camera communication settings to PTP
  • connect camera to mac via usb cable
  • close auto-started application (Apple Photo, Image Capture, etc) then press "Get Shutter Count" button.
