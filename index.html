FreeShutterCounter is OS X application that shows Canon EOS DSLR shutter count. Download: freeshuttercounter-1.1.dmg (1.2 Mb) Current version: 1.1 (22 apr 2016)
Usage: copy app into the /Applications folder. Connect camera to mac via usb cable. Close auto-started application (Apple Photo, Image Capture, etc) then press "Get Shutter Count" button. Application is absolutely free. No charge or payments required. Tested with: Canon EOS 5D Mark III, 5D Mark II, 7D, 60D on: OS X 10.10, OS X 10.11 This app is based on libgphoto2 library.
