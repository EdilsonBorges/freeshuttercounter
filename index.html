FreeShutterCounter is OS X application that shows Canon EOS DSLR shutter count. Download: freeshuttercounter-1.2.1.dmg (1.5 Mb) Changelog: v 1.2.1 (23 dec 2016) - OS X 10.7-10.12 support - updated libs v 1.2 (30 sep 2016) - added OS X 10.9 support - updated libgphoto to 2.5.10 v 1.1 (22 apr 2016)
Usage: - copy app into the /Applications folder - switch camera communication settings to PTP - connect camera to mac via usb cable - close auto-started application (Apple Photo, Image Capture, etc) then press "Get Shutter Count" button. Application is absolutely free. No charge or payments required. Tested with: Canon EOS 5D Mark III, 5D Mark II, 7D, 60D, 50D on: OS X 10.7-10.12 This app is based on libgphoto2 library.
