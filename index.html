FreeShutterCounter is macOS application that shows Canon EOS DSLR shutter count. Download: freeshuttercounter-1.2.2.dmg (1.6 Mb) OS X 10.7-10.12
Changelog: v 1.2.2 (21 jan 2017) - fixed OS X 10.7 error v 1.2.1 (23 dec 2016) - OS X 10.7-10.12 support - updated libs v 1.2 (30 sep 2016) - added OS X 10.9 support - updated libgphoto to 2.5.10 v 1.1 (22 apr 2016)
Usage: - copy app into the /Applications folder - switch camera communication settings to PTP - connect camera to mac via usb cable - close auto-started application (Apple Photo, Image Capture, etc) then press "Get Shutter Count" button. Application is absolutely free. No charge or payments required. Supported cameras: 1D C 1D X 1D Mark IV 7D Mark II 7D 5D Mark III 5D Mark II 6D 70D 60D 50D 700D 650D 600D 550D 500D 100D 1200D 1100D 1000D Unsupported: 1D X Mark II, 5DS, 5DS R, 5D Mark IV, 5D, 80D, 750D, 760D, 1300D This app is based on libgphoto2 library.
